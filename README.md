# STORMAUTH - Using Authtuna, alembic and other examples in future.

## Introduction
This repo is deployed at [https://auth.shashstorm.in](https://auth.shashstorm.in)

The main goal of this repo is to how to use **Authtuna** and how to use its features (examples will be added in other files in the future). 

You can check out the [main.py](https://github.com/shashstormer/stormauth/blob/master/main.py) file to see a basic example of how to use Authtuna with FastAPI.

# Setting Up
### Prerequisites
- Python 3.9 or above.

```bash
pip install authtuna
```

## Running the Application

```bash
python main.py
```

The server will run on `http://localhost:5555`
